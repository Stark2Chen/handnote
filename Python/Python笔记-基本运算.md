# 入门
## 计算
###简单的数学计算

整数和整数相加 得到整数
in
```python
2+2
```
out
```python
4
```

浮点数和浮点数相加 得到浮点数

in
```python
2.5+1.5
```
out
```python
4.0
```

## 给变量进行赋值

赋值格式：<变量名>=<表达式>
in
```python
a=2
a
```
out
```python
	2
```
in
```python
a='hello'
a
```
out
```python
hello
```

##字符串
生成字符串时，单引号与双引号等价

in
```python
s = "hello world"
s
```
out 
```python
'hello world'
```

字符串之间可以进行相加

text.py
```python
b='hello'
c=','
d='world'
e='!'
a=b+c+d+e
print(a)
```
out
	hello,world!
